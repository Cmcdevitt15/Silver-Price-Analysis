# MTH6139 Time Series: Silver Price Predictive Analysis
**Author:** Charles McDevitt (220314286)

## Project Overview
This project utilizes Meta's **Prophet** library in R to analyze and forecast the historical price of the iShares Silver Trust (SLV). The analysis covers data from 2021 through March 2026, capturing post-pandemic market volatility and current macroeconomic shifts.

### Key Analysis Features:
* **Algorithmic Forecasting:** Decomposed silver price action into trend, weekly, and yearly seasonal components.
* **Macroeconomic Event Correlation:** Specifically evaluated the sharp intraday sell-off following the **March 18, 2026, Federal Reserve Meeting** and rising geopolitical tensions in the Middle East.
* **Interactive Visualization:** Implemented interactive time-series explorers using `dygraphs` and `ggplot2`.
